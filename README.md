\# Land Consumption in Lazio



A reproducible geospatial analysis of land consumption in Lazio (Italy) using official ISTAT and ISPRA datasets.



\## Tech stack



\- Docker

\- PostgreSQL

\- PostGIS

\- Python

\- Pandas

\- GeoPandas

\- SQL

\- QGIS



\## Workflow

&#x20;       ↓

QGIS maps
